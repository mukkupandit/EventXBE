## Description

Introducing the EMS Hub - the backend for your Event Management System (EMS) app. Explore and create the latest events effortlessly, whether it's local gatherings or online spectacles. Unlock a myriad of features designed to enhance your event experience and discovery. EMS Hub, your gateway to seamless event management.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
